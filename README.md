# Student Management System (Python)

A simple console-based Student Management System built with Python. This project demonstrates CRUD (Create, Read, Update, Delete) operations using Python lists and a menu-driven interface.

## Features

- Show all students
- Add a new student
- Search for a student
- Update a student's mark
- Delete a student
- Exit the program

## Concepts Used

- Variables
- Lists
- Loops (`for`, `while`)
- Conditional Statements (`if`, `elif`, `else`)
- User Input
- CRUD Operations
- List Methods
  - `append()`
  - `remove()`
  - `index()`
  - `pop()`

## How to Run

1. Make sure Python is installed.
2. Download or clone this repository.
3. Open the project folder.
4. Run:

```bash
python student_management_system_v1.py
```
